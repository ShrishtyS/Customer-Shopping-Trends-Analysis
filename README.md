
# Customer Shopping Trends Analysis

## Project Overview

This project analyzes customer shopping behavior using Python, SQL, Excel and Power BI.

The objective was to identify customer segments, product performance, subscription impact, discount effectiveness and revenue drivers.

## Tools Used

- Python (Pandas)
- SQL
- Power BI

  ## Project Workflow

1. Data Exploration & Cleaning using Python
2. Business Analysis using SQL
3. Interactive Dashboard Development using Power BI
4. Business Insight Generation

## Business Questions

1. Total revenue by gender
2. Discounted customers spending above average
3. Top rated products
4. Shipping type comparison
5. Subscriber vs non-subscriber spending
6. Products with highest discount usage
7. Customer segmentation
8. Top products by category
9. Repeat buyers and subscriptions
10. Revenue contribution by age group
11. Top customer based on spending
12. Discount Influence on spending
13. Preodct with higest Revenue
14. Age group with the highest average spending per purchase

## Key Insights

- Clothing generated the highest revenue.
- Male customers contributed most revenue.
- Loyal customers generated the highest revenue.
- Young Adult customers represented the most valuable demographic segment.
- Subscriber adoption remained relatively low at 27%, indicating potential growth opportunities.
